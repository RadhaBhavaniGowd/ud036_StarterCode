Source code for Movie Trailer Project.
This shows list of movies in html.
You can click on movie image files to show trailer of the specified movie.

STEPS to RUN the application:
* Install Python.
* Add Python to PATH Environmental Variable.
* Next, let's click on the Start Menu and right click My Computer. Click on Advanced System Settings.
* Then click on Environment Variables. This is where we can add Python to the PATH environmental variable.
* Open command prompt and go to the directory where project is cloned.
* Run Command(“python entertainment_center.py”)
